# Il s'agit d'un simple jeu de mord pion en rust

> bon, c'est **pas encore fini** mais ça va venir.

___

## Compilation

c'est un projet `cargo`. Il se compile donc comme tel :
```
cargo build
```

Une fois compilé, le binaire de trouvera dans le dossier `rust-code/target/`.

___
## Lancer le projet en test

Avec cargo, le projet peut aussi être compilé rapidement et lancé directement avec la commande `cargo run`.

___
